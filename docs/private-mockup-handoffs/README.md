# Private mockup handoffs

Owner-facing handoff packages for private race mockup reviews.

Rules for using these handoffs:

- Keep the handoff docs separate from the rendered private mockup UI.
- Steve must approve the handoff before anything is sent to a race director, owner, or prospect.
- Keep outreach respectful and specific; do not shame the current site or promise guaranteed registration growth.
- Use only the approved private review URL, and confirm it remains `noindex,nofollow` before delivery.
